Google

Summer of Code

June 1st, 2020 - August 24, 2020

Google Summer of Code is a two - and a half - month coding program where students all around the world help develop softwares for different open-source organizations. This summer I had the opportunity to work for the organization Gemstash, which is an organization under the Ruby programming language and an application for RubyGems.org. Gemstash is a renowed gem that serves as a private cache and gem server for gems from RubyGems.org. For Summer of Code, I proposed to implement an Amazon S3 cloud storage implementation for gemstash, as well as developing an abstract backend module that would improve the ability to add future backend storage systems. The details of my proposal can be found on the Google Summer of Code website.
